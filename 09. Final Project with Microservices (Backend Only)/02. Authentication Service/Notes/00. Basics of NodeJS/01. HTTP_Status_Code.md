# üü¶ **1xx ‚Äî INFORMATIONAL RESPONSES** (Request Received, Still Processing)

## **100 Continue**

**Definition:**
Server indicates that the initial part of the request has been received and the client may continue sending the rest of the data.

**Example:**
Client begins uploading a large file.
Server replies:

> "Continue sending the file."

---

## **101 Switching Protocols**

**Definition:**
Server agrees to switch to a new protocol as requested by the client (using `Upgrade` header).

**Example:**
Client: ‚ÄúSwitch to WebSocket?‚Äù
Server:

> ‚ÄúSwitching to WebSocket now.‚Äù

---

## **102 Processing**

**Definition:**
Server has accepted the request but is still processing it; no final response available yet.

**Example:**
WebDAV copying many files ‚Üí server sends:

> ‚ÄúProcessing‚Ä¶ don‚Äôt time out.‚Äù

---

## **103 Early Hints**

**Definition:**

**Example:**

> ‚ÄúStart loading CSS and JS early.‚Äù

# üü© **2xx ‚Äî SUCCESS RESPONSES** (Request Completed Successfully)

---

## **200 OK**

**Definition:**
Request succeeded; the response depends on the HTTP method (GET = resource, POST = result info, etc.).

**Example:**
GET `/users` ‚Üí returns list of users.

---

## **201 Created**

**Definition:**
The request succeeded and resulted in the creation of a new resource.

**Example:**

> ‚ÄúUser created with ID: 42.‚Äù

---

## **202 Accepted**

**Definition:**

**Example:**

> ‚ÄúYour video is being processed.‚Äù

---

## **203 Non-Authoritative Information**

**Definition:**

**Example:**

## **204 No Content**

**Definition:**

**Example:**

> ‚ÄúDeleted successfully.‚Äù (empty body)

## **205 Reset Content**

**Definition:**

**Example:**

> ‚ÄúForm submitted. Reset fields.‚Äù

---

## **206 Partial Content**

**Definition:**

**Example:**

# üüß **3xx ‚Äî REDIRECTION RESPONSES** (Further Action Required)

---

## **300 Multiple Choices**

**Definition:**
Resource has multiple possible representations; client should choose.

**Example:**
Movie available in 720p, 1080p, 4K.

---

## **301 Moved Permanently**

**Definition:**
Resource is permanently located at a new URL. Clients should update bookmarks.

**Example:**
`/old-blog` ‚Üí `/blog` permanently.

---

## **302 Found**

**Definition:**
Resource temporarily moved to a different URL.

**Example:**
After login, redirect to `/dashboard`.

---

## **303 See Other**

**Definition:**
Tells client to fetch the resource at another URI using GET, often after POST.

**Example:**
Form submitted ‚Üí
Redirect to `/result` using GET.

---

## **304 Not Modified**

**Definition:**
Resource hasn‚Äôt changed since last request; use cached version.

**Example:**

> ‚ÄúNo new version of image ‚Äî use cache.‚Äù

---

## **307 Temporary Redirect**

**Definition:**
Temporary redirect; **same HTTP method must be used**.

**Example:**
POST `/login` ‚Üí temporarily redirect ‚Üí must stay POST.

---

## **308 Permanent Redirect**

**Definition:**
Same as 301, but preserves HTTP method.

**Example:**
POST `/v1/users` ‚Üí permanently moved to `/v2/users`.

# üü• **4xx ‚Äî CLIENT ERROR RESPONSES** (Client Made a Mistake)

## **400 Bad Request**

**Definition:**

**Example:**

## **401 Unauthorized**

**Definition:**

**Example:**

> ‚ÄúPlease log in.‚Äù

## **402 Payment Required**

**Definition:**

---

## **403 Forbidden**

**Definition:**

**Example:**

---

## **404 Not Found**

**Definition:**
Requested resource does not exist.

**Example:**

## **405 Method Not Allowed**

**Definition:**

**Example:**

---

## **406 Not Acceptable**

**Definition:**

**Example:**

## **407 Proxy Authentication Required**

**Definition:**

## **408 Request Timeout**

**Definition:**

---

## **409 Conflict**

**Definition:**

---

## **410 Gone**

**Definition:**

**Example:**
Old page intentionally deleted.

---

## **411 Length Required**

**Definition:**

---

## **412 Precondition Failed**

## **413 Payload Too Large**

---

## **415 Unsupported Media Type**

**Definition:**

**Example:**

---

## **416 Range Not Satisfiable**

**Definition:**

---

## **417 Expectation Failed**

**Definition:**

**Definition:**

## **422 Unprocessable Entity**

**Definition:**

**Example:**

---

## **429 Too Many Requests**

**Definition:**

**Example:**

> ‚ÄúTry again later.‚Äù

## **431 Request Header Fields Too Large**

# üü• **5xx ‚Äî SERVER ERROR RESPONSES** (Server Made a Mistake)

## **500 Internal Server Error**

**Definition:**
Generic server error; something unexpected went wrong.

**Example:**

---

## **501 Not Implemented**

**Definition:**
Server does not support the requested functionality/method.

**Example:**

---

## **502 Bad Gateway**

**Definition:**
Server (acting as gateway/proxy) received invalid response from upstream server.

**Example:**
Load balancer cannot reach backend.

---

## **503 Service Unavailable**

**Definition:**
Server temporarily overloaded or under maintenance.

**Example:**
Server rebooting ‚Üí

> ‚ÄúTry later.‚Äù

---

## **504 Gateway Timeout**

**Definition:**

**Example:**

---

## **505 HTTP Version Not Supported**

**Definition:**
HTTP version used in request is not supported.

---

## **506 Variant Also Negotiates**

**Definition:**

---

## **507 Insufficient Storage**

**Definition:**
Server cannot store the needed representation.

---

## **508 Loop Detected**

**Definition:**

---

## **510 Not Extended**

**Definition:**
Request lacks required extensions for processing.

---

## **511 Network Authentication Required**

**Definition:**
Client must authenticate to gain network access.

**Example:**
Captive Wi-Fi login page.
