## ðŸŽ´ Flashcard Set: SQL Basics

**Q:** What does SQL stand for?
**A:** Structured Query Language.

---

**Q:** What kind of databases use SQL?
**A:** Relational databases.

---

**Q:** What is a schema?
**A:** A fixed structure that defines columns, data types, and rules.

---

**Q:** What is a primary key?
**A:** A unique identifier for a row.

---

**Q:** What is a foreign key?
**A:** A key linking one table to another.

---

**Q:** What is ACID?
**A:** Atomicity, Consistency, Isolation, Durability (transaction safety).

---

**Q:** What is a JOIN in SQL?
**A:** Operation to combine data from two tables.

---

**Q:** Example of SQL command to retrieve all rows?
**A:** `SELECT * FROM table_name;`

---

## ðŸŽ´ Flashcard Set: NoSQL Basics

**Q:** What does NoSQL stand for?
**A:** Not Only SQL.

---

**Q:** What is the main feature of NoSQL?
**A:** Flexible, schema-less data storage.

---

**Q:** Name four types of NoSQL databases.
**A:** Document, Key-Value, Wide-Column, Graph.

---

**Q:** Example of a Document DB?
**A:** MongoDB.

---

**Q:** Example of a Key-Value DB?
**A:** Redis.

---

**Q:** Example of a Graph Database?
**A:** Neo4j.

---

**Q:** What is horizontal scaling?
**A:** Adding more servers to increase capacity.

---

**Q:** NoSQL often uses which consistency model?
**A:** Eventual consistency.

---

## ðŸŽ´ Flashcards: SQL vs NoSQL Comparison

**Q:** SQL schema type?
**A:** Fixed schema.

---

**Q:** NoSQL schema type?
**A:** Schema-less.

---

**Q:** SQL scaling type?
**A:** Vertical scaling.

---

**Q:** NoSQL scaling type?
**A:** Horizontal scaling.

---

**Q:** SQL best for?
**A:** Financial systems, structured data, transactions.

---

**Q:** NoSQL best for?
**A:** Big data, social feeds, flexible unstructured data.

---

**Q:** SQL handles relationships using?
**A:** Joins + foreign keys.

---

**Q:** NoSQL handles relationships using?
**A:** Embedded documents or graph nodes/edges.
