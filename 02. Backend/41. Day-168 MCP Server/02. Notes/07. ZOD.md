**Zod** is a **TypeScript-first schema declaration and validation library**.

* It allows developers to **define the structure and type of data** in a declarative way.

### 2ï¸âƒ£ Basic Usage

```ts
import { z } from "zod";

// Define a schema
const userSchema = z.object({
  name: z.string(),
  age: z.number().min(0),
  email: z.string().email()
});

// Validate data
const result = userSchema.safeParse({
  name: "Alice",
  age: 25,
  email: "alice@example.com"
});

if (result.success) {
  console.log("Valid data:", result.data);
} else {
  console.error("Validation errors:", result.error.format());
}
```

**Explanation:**

* `z.object({...})` â†’ defines an object schema.
* `z.string()`, `z.number()`, `z.string().email()` â†’ define individual field types.
* `.safeParse()` â†’ validates data **without throwing**, returning `{ success: boolean, data?, error? }`.
