
### ‚úÖ **Topic**: Exponentiation by Squaring (Efficient Power Calculation)

---

### üìå **Purpose**
Efficiently compute `x^n` (x raised to the power n) using a logarithmic approach. It works for both positive and negative integer powers.

---

### üìÑ **Code**
```js
function pow(x, n) {
    if (n == 0) return 1;

    let m = n;
    if (n < 0) {
        n = -n;
    }

    let ans = 1;
    while (n > 0) {
        if (n % 2 == 0) {
            x = x * x;
            n = Math.floor(n / 2);
        } else {
            ans = ans * x;
            n = n - 1;
        }
    }

    if (m < 0) {
        ans = 1 / ans;
    }

    return ans;
}
```

---

## üß† **Step-by-Step Execution (Example: pow(2, 10))**

### üîπ Step 1: Base Case
```js
if (n == 0) return 1;
```
- Any number to the power 0 is 1.
- If `n = 0`, function immediately returns `1`.

---

### üîπ Step 2: Store original `n` in `m`
```js
let m = n;
```
- We store the original `n` to later check if it was negative.  
  This helps us apply `1 / ans` if `n` was negative.

---

### üîπ Step 3: Make `n` Positive
```js
if (n < 0) {
    n = -n;
}
```
- If `n` is negative, we work with its absolute value temporarily.
- For example: `pow(2, -5)` becomes `pow(2, 5)`, and at the end we return `1 / ans`.

---

### üîπ Step 4: Initialize `ans`
```js
let ans = 1;
```
- `ans` will store the final result.
- It is initialized as 1 (multiplicative identity).

---

### üîÑ Step 5: Loop until `n` becomes 0
```js
while (n > 0) {
```

This loop performs efficient power calculation:
- Reduces the number of multiplications from **O(n)** to **O(log n)**.
- Works by using properties of even and odd powers:
  - If `n` is even:  
    - `x^n = (x^2)^(n/2)`
  - If `n` is odd:  
    - `x^n = x * x^(n-1)`

---

### ‚öôÔ∏è Loop Breakdown

#### ‚ñ∂Ô∏è First Iteration (`n = 10`, even)
```js
x = x * x;     // x = 2 * 2 = 4
n = n / 2;     // n = 10 / 2 = 5
```

#### ‚ñ∂Ô∏è Second Iteration (`n = 5`, odd)
```js
ans = ans * x; // ans = 1 * 4 = 4
n = n - 1;     // n = 5 - 1 = 4
```

#### ‚ñ∂Ô∏è Third Iteration (`n = 4`, even)
```js
x = x * x;     // x = 4 * 4 = 16
n = n / 2;     // n = 4 / 2 = 2
```

#### ‚ñ∂Ô∏è Fourth Iteration (`n = 2`, even)
```js
x = x * x;     // x = 16 * 16 = 256
n = n / 2;     // n = 2 / 2 = 1
```

#### ‚ñ∂Ô∏è Fifth Iteration (`n = 1`, odd)
```js
ans = ans * x; // ans = 4 * 256 = 1024
n = n - 1;     // n = 1 - 1 = 0
```

‚úÖ Loop ends since `n = 0`

---

### üîô Step 6: If `n` was negative, return reciprocal
```js
if (m < 0) {
    ans = 1 / ans;
}
```
- If the original power was negative, we return `1 / ans`.

---

### üü¢ Step 7: Return final result
```js
return ans;
```

---

### üí° **Time Complexity**:  
- `O(log n)` ‚Äî Efficient due to squaring and halving.
