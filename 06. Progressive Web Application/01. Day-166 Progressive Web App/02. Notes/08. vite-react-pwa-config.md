### 2ï¸âƒ£ Workbox / Caching

```js

    {
      urlPattern: "*",
```

* **Workbox** â†’ Library for caching and offline support in service workers.
* **runtimeCaching** â†’ Rules for handling network requests **after the SW is installed**.
* **urlPattern: "\*"** â†’ Matches all requests.
* **handler: "CacheFirst"** â†’ Strategy:

  * First check **cache** for a resource.
  * If not in cache â†’ fetch from network.
  * Good for static assets (JS, CSS, images).

> Note: `"*"` as a pattern may cache too broadly. For APIs, usually `NetworkFirst` is better.
